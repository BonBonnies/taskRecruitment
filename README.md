# GitHub Repository Viewer

Spring Boot application that fetches GitHub repositories for a given user.

## Requirements

- Java 21
- Gradle

## How to Run

1. Clone the repository
2. Navigate to the project directory
3. Run using  Gradle:
   ```bash
   ./gradlew bootRun
   ```
4. The application will start on port 8080
